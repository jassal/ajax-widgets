Intalio|Ajax Widgets were primarily created to be used within Intalio|Designer application.  The widgets allow users to quickly create web based forms for use within a business process.

h2. Key Features

* Fieldset object for easy form section creation.
* Integrated help image with tooltip.
* Integrated validation image and text message.
* Automatic required indicator.
* Intelligent drag and drop.
