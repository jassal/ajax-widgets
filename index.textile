Intalio|Ajax Widgets were primarily created to be used with Intalio's Designer application.  The widgets allow users to quickly create web based forms for use within a business process.

For a quick introdoction to using the widgets see the "One Minute Form":one_minute_form.html tutorial.

h2. Key Features

* Fieldset object for easy form section creation.
* Integrated help image with tooltip.
* Integrated error image and text message.
* Integrated 'required' indicator.
* Intelligent drag and drop.

Intalio|AJAX widgets are designed primarily to be used within a Fieldset object object.  This is not required and they work fine outside of a Fieldset, but you get some advanced functionality when doing so.
